# 通用字段

字段名称默认值字段解释数据类型是否必填可选值备注
type图形类型string是"line" "circle" "arc" "polygon" "rect" "text" "sector" "image" "polyline" "bezierCurve"
strokenull图形轮廓颜色string否颜色值
fill"#000"图形填充颜色string否颜色值
fillOpacity1图形轮廓颜色透明度number否取值范围 0-1
strokeOpacity1图形填充颜色透明度number否取值范围 0-1
zlevel0图形层级number否
draggablefalse图形是否可以拖拽boolean否
params{}给图形添加的额外参数boolean否对图点击等操作需要拿原数据的时候非常有用

# 1. 直线

字段名称默认值字段解释数据类型是否必填可选值备注
type"line"图的类型string是"line"
x10起始点横坐标number是
y10起始点纵坐标number是
x20终止点横坐标number是
y20终止点纵坐标number是
lineWidth1线条宽度number否
lineDash"solid"线条类型string否"solid"(实线) "dashed"(虚线) "dotted"(点虚线)

# 示例

{
  "type": "line",
  "x1": 0,
  "y1": 0,
  "x2": 100,
  "y2": 100,
  "stroke": "red"
}

  "r": 100,
